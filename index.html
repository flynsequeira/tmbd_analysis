Introduction¶

This data set contains information about 10,000 movies collected from The Movie Database (TMDb), including user ratings and revenue. Certain columns, like ‘cast’ and ‘genres’, contain multiple values separated by pipe (|) characters. We need to solve that, fix in the data wrangling phase. We also put some of the data in the right format, add a new column etc We look into questions like, how the runtime of movies have changed over the years, when do movies usually make more money in revenue and also check how each popular genres have faired through the years.

def string_to_series(st):
    if type(st).__name__ == 'str':
        return st.split('|')
    else:

        return []

# Put in python date format. Source: Udacity Course on intro to data analyst

def parse_date(date):
    if date == '':
        return None
    else:
        # date is in mm/dd/yy format
        return dt.strptime(date, '%m/%d/%y')

# A function to group the data by a field and then take the average of another field for that perticular selected group.
def group_and_mean_of_popular(dataf,groups_col,average_col):
    all_group_set = list(dataf[groups_col].unique())
    all_group_set.sort()
    output_list = []
    for each_group in all_group_set:
#         Popularity checked here
        new_df = dataf[dataf[groups_col]==each_group]
        avg_val = sum(new_df[average_col])/float(len(new_df[average_col]))
        output_list.append([each_group,avg_val])
#     print output_list
    output_df = pd.DataFrame(output_list,columns=[groups_col,average_col])
    return output_df

Conclusions¶

Runtime vs year of release¶

There might be a possibility that movies are getting shorter over the years. Especially lately in the last 15 years. The data is noisy pre-90s and is mostly constant. So we focused on the post 90s era. We see a steady pattern of how movies are getting shorter. It might be a good thing to see well packed movies in a shorter time.

Preference of month of release¶

There is a clear pattern on when movies are released. This doesn't imply causation that movies make more money during summers and december. There could be many factors affecting this. But the difference in revenue is significant.

Genre Preference¶

For the last 10 years we have seen that most movie types have had the same revenue. But few exceptions to this are Adventure, comedy, Thrill, Action etc. This could be possible becacuse of the budget that is put into action and adventure movies. They have gotten significantly more beautiful in the recent years due to amazing innovation in the effects department.

All of these show correation between various fields. Runtime over the years has shown a drop. But this doesn't show any statistical significance. Similarly month of release doesn't cause movies to generate high revenue, and making a movie of a particular genre wont cause a movie to get high revenue.
